# 분야별 한 문장 정리 (from programmers)

* 탐욕법 (Greedy) : 부분적인 최적해가 전체적인 최적해가 되는 마법!
  * 언제나 통하지는 않지만, 이런 방법이 통하는 문제들을 알아보자

* DFS & BFS : 깊이/너비 우선 탐색을 사용해 원하는 답 찾아보자

* 스택/큐 : LIFO, FIFO, push & pop! 스택과 큐를 이용해서 문제를 풀어보자
